导入说明

支付宝导出：支付宝 → 我的 → 账单 → 右上角「···」→「开具交易流水证明」→「用于个人对账」→ 选时间范围 → 确认 → 下载（解压密码为身份证后 6 位）

微信导出：微信 → 我 → 服务 → 钱包 → 账单 → 右上角「···」→「常见问题」→「下载账单」→ 选「用于个人对账」（不要选「下载交易流水证明」，那个是动态 HTML 页面，无法直接解析）→ 选时间范围 → 确认 → 下载

提示：微信导出的文件后缀是 .xlsx，实际是 CSV 格式，直接导入即可。支付宝推荐用 Chrome/Edge 浏览器下载以避免乱码。

⚠️ 若导入为 0 条记录：
· 微信：必须选「用于个人对账」，「下载交易流水证明」是 HTML 壳子，没有原始数据
· 支付宝：ZIP 解压密码为身份证后 6 位
· 也可以在手机上下载账单后，复制文件内容粘贴到「粘贴文本」框导入
